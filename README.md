## âœ¨ Features

### Design & UI

- **Modern Aesthetic**: Clean, professional design with vibrant orange accent color
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll-triggered animations using Intersection Observer API
- **Floating Hero Cards**: Eye-catching animated cards in the hero section
- **Dark Theme Sections**: Strategic use of dark backgrounds for visual contrast

### Navigation

- **Sticky Header**: Fixed navigation with hide-on-scroll-down, show-on-scroll-up behavior
- **Smooth Scrolling**: Seamless navigation to page sections
- **Active Link Highlighting**: Current section highlighted in navigation
- **Mobile Menu**: Hamburger menu for mobile devices with auto-close on link click

### Sections

1. **Hero Section**: Dynamic floating cards with statistics and user engagement
2. **Statistics**: Key metrics displayed with animated counters
3. **Our Story**: Company background and mission
4. **Why Choose Us**: Benefits and value propositions
5. **Services**: Grid layout of service offerings
6. **Our Approach**: Interactive tabs with Mission/Vision content
7. **Growth Business**: Service cards with detailed offerings
8. **Win on Amazon**: Dark-themed section highlighting Amazon strategies
9. **Contact/Booking**: Calendar integration placeholder with contact information
10. **Footer**: Multi-column layout with navigation and contact details

### Animations

- **Scroll Animations**: 5 different animation types (fadeInUp, fadeInLeft, fadeInRight, fadeIn, scaleIn)
- **Staggered Delays**: Sequential animation timing for grouped elements
- **Hero Card Entrance**: Slide-in animations with rotation
- **Floating Effects**: Continuous subtle movement on hero cards
- **Performance Optimized**: Uses Intersection Observer for efficient scroll detection

## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local server for development (Live Server, Python HTTP server, etc.)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/alamin-islam0/frontend-interview-task-sm-tech.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd frontend-interview-task-sm-tech
   ```

3. **Open in browser**
   - Simply open `index.html` in your browser, or
   - Use a local development server:

     ```bash
     # Using Python 3
     python -m http.server 8000

     # Using Node.js http-server
     npx http-server
     ```

4. **View the site**
   - Open `http://localhost:8000` in your browser
